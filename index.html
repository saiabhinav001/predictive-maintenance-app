Platform Documentation

Platform Overview

MachineInsight Pro is a self-contained, enterprise-grade predictive maintenance platform. It uses a sophisticated machine learning model to analyze real-time sensor data and predict equipment failures. The core model was identified using automated machine learning (AutoML) techniques to ensure high performance and then deployed as part of this self-hosted application, enabling proactive maintenance strategies that reduce downtime and operational costs.

Technology Stack & Architecture

Frontend & UI

  • Modern HTML5 & Tailwind CSS
  • GSAP for fluid animations
  • Chart.js for data visualization
  • Responsive glass morphism design

Backend & AI Model

  • Python & Flask for the API backend
  • LightGBM Classifier as the core predictive model
  • Scikit-learn for data preprocessing
  • Vercel Serverless Functions for scalable deployment

Model Performance & Limitations

Strengths

  • High Accuracy: The LightGBM model is known for its high performance, achieving an F1-score of over 99% in original tests, ensuring excellent precision and recall.
  • Optimized Performance: Leverages a gradient boosting framework that is fast and efficient, making it suitable for real-time prediction scenarios.
  • Comprehensive Detection: Highly effective at identifying the most common and critical failure types, including Power Failure and Tool Wear.

Limitations

  • Data Dependency: Accuracy is tied to the training data. Performance may vary on equipment with significantly different operating characteristics.
  • Rare Event Prediction: May have lower accuracy on extremely rare or novel failure types not well-represented in the training dataset.
  • Model Drift: As equipment ages or conditions change, performance can degrade. Periodic retraining with new data is recommended.
