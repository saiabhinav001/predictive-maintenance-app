Platform Documentation

Platform Overview

MachineInsight Pro is an enterprise-grade predictive maintenance platform that leverages an advanced, automatically generated machine learning model to analyze real-time sensor data. Our AI, selected and optimized by IBM's AutoAI, can predict potential failures with exceptional accuracy, enabling proactive maintenance strategies that reduce downtime and operational costs.

Technology Stack & Architecture

Frontend Technologies

  • Modern HTML5 with semantic markup
  • Tailwind CSS for responsive design
  • GSAP for advanced animations
  • Chart.js for data visualization
  • Glass morphism UI design system

Backend & AI

  • IBM Watson AutoAI for automated model generation
  • LightGBM (LGBM) Classifier (Top-performing model)
  • Automated Hyperparameter Optimization (HPO)
  • Automated Feature Engineering
  • Vercel Serverless Functions (Backend Proxy)

Model Performance & Limitations

Strengths

  • 99.4% F1-weighted score (industry-standard metric)
  • Rapid Model Development: AutoAI generated and ranked 9 model pipelines in under 12 minutes.
  • Automated Optimization: The best pipeline includes automated feature engineering and hyperparameter tuning.
  • Excellent performance on common failure modes.

Limitations

  • Model performance is dependent on data quality.
  • Reduced accuracy on extremely rare or random failures.
  • Continuous monitoring and retraining are needed for new equipment types or changing conditions.
